# 20 Problem Space 作者契约

## 契约范围

本页裁定 accepted problem model 应如何在本目录落位。

## Allowed Objects

- domain model
- ontology
- givens / business rules
- accepted environment assumptions
- problem-side structural distinctions

## Disallowed Objects

- requirements
- architecture views
- candidate assumptions
- accepted decision records

## Dependency Discipline

- 本目录以前置 positioning 为 framing 上游
- 本目录不应反向依赖 requirements 才成立
- 若某对象主要表述的是“世界应该怎样”，应迁入 `40_requirements/`
