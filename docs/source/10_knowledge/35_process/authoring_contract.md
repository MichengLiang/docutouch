# 35 Process 作者契约

## 契约范围

本页裁定 accepted process architecture 应如何在本目录落位。

## Allowed Objects

- accepted process architecture
- accepted activity model
- accepted evaluation coordinates
- accepted grounding / repair obligation
- accepted 回流 / 共演化规则

## Disallowed Objects

- operational runbook
- open worklist
- migration / audit / coverage record
- unresolved process proposal

## Dependency Discipline

- `process/` 以前置 positioning、problem-space 与 principles 为 accepted framing 上游
- `process/` 不得退化成运行操作家族的上位别名
- 若对象主要记录的是阶段状态、完成度或迁移轨迹，应迁入 `30_records/`
- 若对象仍处于候选与未收敛状态，应迁入 `20_deliberation/`
