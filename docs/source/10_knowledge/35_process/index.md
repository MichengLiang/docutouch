# 35 Process

## 作用域

本目录承载 accepted knowledge tree 中的过程骨架家族。

它回答：

- 某个成熟领域的 accepted process architecture 是什么；
- 活动轴、评估坐标、交互义务与回流规则如何作为 accepted knowledge 存在；
- 哪些 process-side judgments 仍属于现行知识，而不是 records 或开放 deliberation。

## 典型对象

- activity architecture
- process obligations
- evaluation coordinates
- co-evolution rule
- grounding / repair obligation

## 不承担的职责

- 不写系统运行操作手册
- 不写 open worklist
- 不写历史状态记录
- 不写 reader-facing guide surface
