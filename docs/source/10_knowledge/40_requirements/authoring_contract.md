# 40 Requirements 作者契约

## 契约范围

本页裁定 accepted requirements objects 的落位边界。

## Allowed Objects

- accepted requirement
- accepted need / goal
- constraint
- acceptance condition
- trace anchor

## Disallowed Objects

- givens / business rules
- architecture views
- accepted decision record
- candidate requirement / candidate spec

## Dependency Discipline

- requirements 以前置 positioning、problem-space 与 principles 为上游
- requirements 不应以 architecture 为 authority 前提
- 若对象主要在解释“为什么选 A 不选 B”，应迁入 `60_decisions/`
