# 70 Interfaces 作者契约

## 契约范围

本页裁定 accepted interface / contract surface objects 的落位边界。

## Allowed Objects

- interface contract
- protocol surface definition
- schema
- consumer contract layer

## Disallowed Objects

- architecture rationale
- candidate interface design
- operational usage workaround
- migration record

## Dependency Discipline

- interfaces 以前置 principles、requirements 与 architecture 为 authority 上游
- interfaces 不应反向定义 architecture 的主体结构
- 若某页主要在解释“为什么如此裁决”，应迁入 `60_decisions/`
