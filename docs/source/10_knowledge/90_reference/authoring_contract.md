# 90 Reference 作者契约

## 契约范围

本页裁定 accepted knowledge tree 中哪些支撑性页面应进入 `reference/`。

## Allowed Objects

- glossary
- concept inventory
- notation guide
- appendix-like support page
- crosswalk / reference table

## Disallowed Objects

- 主体对象的唯一 authority 定义
- 未收敛术语草案
- migration / inventory record

## Dependency Discipline

- `reference/` 可以派生整理其他主体家族对象
- `reference/` 不应反向成为主体家族的唯一 authority source
- 所有派生性支撑页都必须显式回指其 canonical subject-family source
- 若某对象其实在定义 accepted principle、requirement、architecture 或 interface，应迁回对应主体家族
