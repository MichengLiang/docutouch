# 10 Issues 作者契约

## 契约范围

本页裁定哪些未决对象应进入 `issues/`。

## Allowed Objects

- unresolved issue
- unresolved boundary question
- unresolved classification question
- unresolved decision point

## Disallowed Objects

- proposal
- assumption
- candidate specification
- worklist item

## Dependency Discipline

- issue 页应显式指向其相关 accepted family 或 target surface
- issue 不应膨胀为把 proposal、gap、conflict 都塞进去的总页
- 若页的核心变为“哪种方案更好”，应迁入 `20_proposals/`
- 若页的核心变为“哪些说法互相打架”，应迁入 `50_conflicts/`
