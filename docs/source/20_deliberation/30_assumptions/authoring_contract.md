# 30 Assumptions 作者契约

## 契约范围

本页裁定哪些桥接性前提应进入 `assumptions/`。

## Allowed Objects

- provisional assumption
- bridge premise
- temporary simplification needed to move work forward

## Disallowed Objects

- accepted principle
- accepted given
- issue statement
- generic evidence gap

## Dependency Discipline

- assumption 页必须显式说明其服务于哪个 issue、proposal 或 candidate spec
- assumption 不应被提前写成 accepted principle
- 若 assumption 已被正式确认，其宿主应迁回 `10_knowledge/`
