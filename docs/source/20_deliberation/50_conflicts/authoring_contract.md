# 50 Conflicts 作者契约

## 契约范围

本页裁定哪些显式不兼容对象应进入 `conflicts/`。

## Allowed Objects

- explicit conflict
- explicit incompatibility between candidate or accepted objects
- trade-off collision

## Disallowed Objects

- generic unresolved issue
- plain proposal set
- missing evidence note

## Dependency Discipline

- conflict 页必须显式指出冲突双方或冲突对象群
- 若页的中心只是“还有问题没定”，应迁回 `10_issues/`
- 若页的中心只是“缺少 backing”，应迁回 `60_evidence_gaps/`
